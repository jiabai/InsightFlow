# 10 — 错误与边界

Status: completed

## Parent

`.scratch/insightflow-v1/PRD.md`

## What to build

漫式阅读全链噜错误处理与边界状态覆盖，确保产品在异常场景下的良好体验。

- 加载态：内容提取中、上传中、问题生成中三个阶段的不同骨架屏动画
- 空状态：提取内容不足、生成零问题、无 Tag 时的引导提示
- 错误态：网络断开、后端超时、LLM 调用失败、处理失败、研究会话失败的降级和重试
- 边界：极长文章（> 100K 字）、无标题结构、纯图片页面、纯代码页面
- 退出安全：任何状态下 ESC 或关闭都能完整恢复原始页面

## Acceptance criteria

- [ ] 提取/上传/生成三个阶段各有不同的骨架屏动画
- [ ] 网络断开时显示"网络连接失败"并自动重试（3 次指数退避）
- [ ] Content Status = Failed 时显示错误原因和"重新处理"按钮
- [ ] 研究会话失败时自动降级为纯原文回答，提示"已降级为基础回答"
- [ ] 空内容提取提示"未检测到可提取的内容"
- [ ] 问题生成为 0 时提示"本文未生成问题，可能内容过短"
- [ ] 所有错误场景中 ESC 或关闭都能正常退出并恢复页面
- [ ] 极长文章（100K+ 字）不导致浏览器卡死（分块处理有超时保护）
- [ ] 关闭页面/切换标签页时正确处理未完成的操作
