# 09 — 扩展 - 沉浸式阅读全链路串联

Status: ready-for-agent

## Parent

`.scratch/insightflow-v1/PRD.md`

## What to build

将沉浸式阅读视图与后端全链路串联，实现从点击扩展到流式问答的完整用户体验。

- 沉浸式阅读侧边栏集成问题列表（按 Tag 分组、可折叠）
- 点击列表中的 Question → 触发流式问答 → 答案在问题下方内联展开，打字机效果
- 已答问题旁显示 ✓ 标记，缓存答案避免重复请求
- 阅读进度追踪：滚动原文时高亮当前段落对应的问题
- 上传触发集成：沉浸式阅读中自动/手动触发内容上传和问题生成
- user_id 通过 SHA-256 自动生成并持久化到 `browser.storage.sync`

## Acceptance criteria

- [ ] 从点击扩展到看到问题列表的完整流程可走通
- [ ] 问题按 Tag 分组展示，点击 Tag 标题可折叠/展开
- [ ] 点击问题后答案在问题下方内联展开，打字机效果
- [ ] 同一 Session 内再次点击已回答问题直接显示缓存答案
- [ ] 滚动原文时当前段落对应的问题在侧边栏中高亮
- [ ] 后端异常时前端显示友好错误提示（不崩溃）
- [ ] user_id 在浏览器重启后保持不变
- [ ] `#question-sidebar` 在进入和退出沉浸模式时正确创建和销毁
