# 03 — 沉浸式阅读

Status: ready-for-agent

## Parent

`.scratch/insightflow-v1/PRD.md`

## What to build

InsightFlow 的核心用户界面——沉浸式阅读模式。用户点击扩展图标 → 提取内容 → 立即进入此模式，是整个产品的第一体验入口。

- 创建全屏覆盖层替换原始网页，左侧渲染提取的文章内容（Markdown），右侧预留问题侧边栏占位
- 关闭按钮（×）或 ESC 键退出沉浸模式，完整恢复原始页面 DOM
- 样式：清晰排版（合理字体、行间距、宽度限制），浅蓝背景（`#e6f7ff`）
- 内容少于 500 字符时不进入沉浸模式，直接提示用户
- 支持深色/浅色主题切换
- 后续切片会在此视图上叠加问题侧边栏和问答功能

## Acceptance criteria

- [ ] 页面正确替换为全屏阅读视图，左侧原文、右侧预留侧边栏区域
- [ ] 原文以清晰排版渲染，Markdown 正确解析
- [ ] ESC 键或关闭按钮退出后，原始页面 DOM 完全恢复，不触发刷新或跳转
- [ ] 内容不足 500 字符时给出明确提示
- [ ] 在 1920×1080 和 1366×768 分辨率下布局正常
- [ ] 沉浸式容器 z-index 足够高，不被原页面元素遮挡
